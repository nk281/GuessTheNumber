# GuessTheNumber
# 🎯 Number Guesser Game

A fun and interactive browser-based game where the player tries to guess a randomly generated secret number within a minimum number of attempts.

---

## 🕹️ How to Play

1. The game generates a **random secret number** within a defined range (1–100).
2. Enter your guess in the input field and click **"Guess"**.
3. The game will tell you if your guess is **too high**, **too low**, or **correct**.
4. Keep guessing until you find the number!

---

## ✨ Features

- 🎲 Random number generated on every new game
- 📢 Instant feedback — "Too High", "Too Low", or "Correct!"
- 🏆 Win/Loss messages at the end of each game
